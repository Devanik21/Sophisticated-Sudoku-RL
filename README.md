# Sophisticated-Sudoku-RL
 MCTS with PUCT - Uses the same MCTS that beat world champions in Go and Chess, adapted for Sudoku with the PUCT (Predictor + UCT) formula Deep Q-Learning (DQN) - Prioritized experience replay learns from past solving attempts, focusing on the most informative experiences Constraint Propagation - CSP (Constraint Satisfaction Problem)
